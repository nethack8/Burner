# Burner
## Shell script that uses dd to delete files

Requirements: 
  - dd
  
Installation: 
  git clone https://github.com/nethack8/Burner.git
  cd Burner
  chmod 700 burner.sh
  
  Or use this oneliner:
  git clone https://github.com/nethack8/Burner.git; cd Burner; chmod 700 burner.sh
Usage: 
  ./burn.sh \<file\>
